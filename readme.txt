# 🚦Smart Traffic License Verification System

This futuristic application allows **real-time driving license verification** using **CNIC scanning via webcam or manual entry**. It helps traffic authorities verify a citizen's license on the spot and provides a **7-day grace period** if the license is not found.

## 🌟 Features

- 🔍 OCR-based CNIC extraction via webcam or image upload
- 🕒 7-day grace period for expired/missing licenses
- 📄 CNIC scan history
- 📷 Manual CNIC entry option
- ⚡ Neon-styled futuristic UI
- 🧠 Built with Python, Streamlit, OpenCV & Tesseract OCR
